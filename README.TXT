

PROJECT TITLE: Dice Roller
HOW TO START THIS PROJECT:To be able to start this project you need to download greenfoot and java,
run the file "project.greenfoot" and hit run.
AUTHOR: Alejandro Gonzalez	
USER INSTRUCTIONS: This project is an automatic dice roller, where you are able to decide how many sides 
does the dice have. Based on that the program will store the value obtained by rolling the two dice.
You can roll the dice once at a time or roll them ten times by hitting "rollx10".
You can use the sort value to be able to sort each of the values that are possible to achieve by the
number of sides of the dice, in a greater ot lower sort.
